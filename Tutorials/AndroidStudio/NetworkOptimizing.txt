因为一些Google服务和Android.com被屏蔽，所以需要做些设置，才能让Android Studio的网络加载更流畅

1. 修改Hosts，加入下面几行

203.208.46.146 www.google.com
203.208.46.146 dl.google.com
203.208.46.146 dl-ssl.google.com

2. 修改Android SDK Manager设置

打开Android SDK Manager -> Tools -> Options

IPv4: http://ubuntu.buct.edu.cn/ 端口：80
IPv4: http://ubuntu.buct.cn/ 端口：80

选中  Force https://... srouces to be fetched using http://...


Mac下的Android Studio包下载时只是个开发工具，不带SDK包，启动时长时间下载真的受不了，以下是优化步骤

1. 禁用首次启动检测

找到AndroidStudio应用程序，右击程序图标->显示包内容->使用文本编辑器打开Contents/bin/idea.properties文件，增加一行

disable.android.first.run=true

2. 设置代理

启动Android Studio，在欢迎界面，Configure -> 打开SDK Manager，别管加载成功和失败，切换一下窗口，让SDK Manager的主菜单出来
点击菜单Android SDK Manger -> Preferences -> 设置如下代理：

IPv4: http://ubuntu.buct.edu.cn/ 端口：80
IPv4: http://ubuntu.buct.cn/ 端口：80

更多代理见 http://www.androiddevtools.cn/

设置代理时，记得Clear Cache
记得选中 Force https://... srouces to be fetched using http://...

3. 重启，也可以把你的禁用启动检测关掉
